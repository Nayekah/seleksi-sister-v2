## How to Run

### Requirements
- cisco packet tracer

<br/>

### Running the Application
1. Clone this repo
   ```bash
   https://github.com/Nayekah/seleksi-sister-v2.git
   ```

2. Navigate to the cloned repository.
   ```bash
   cd seleksi-sister-v2
   cd "Sybau Labubu Tracer"/
   ```
   
3. Open cisco packet tracer, and load sister.pkt
